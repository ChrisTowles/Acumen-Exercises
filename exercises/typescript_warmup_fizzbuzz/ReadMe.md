# FizzBuzz

## Exercise Info
- **Skills Tested:** Basic loops, conditionals, modulo operator
- **Success Criteria:** All tests pass

## Problem
Implement the classic FizzBuzz:
- For numbers 1-n, return array where:
  - Multiples of 3: "Fizz"
  - Multiples of 5: "Buzz"
  - Multiples of both: "FizzBuzz"
  - Otherwise: the number as string

## Running Tests

```bash
npm install
npm run dev          # run tests once
npm run dev:watch    # watch mode
```
